hexo ブログ導入まとめ(Windows PC)

Windows PCにてhexo + GitHub pagesでブログを公開しましたので、導入方法をまとめます。

GitHubアカウントの作成

GitHubにサインアップします。アカウントを作成し、リポジトリを作りましょう。

https://github.com/

PCにGit for Windows を導入

Git(バージョン管理ソフトウェア)をインストールします。

https://gitforwindows.org/

GitHubとの連携にも使用しますし、後ほど導入するhexoの各種コマンドはGit Bashと呼ばれるターミナルで操作します。

※Macなどと違い、Windowsコマンドプロンプトはイマイチ操作しにくいので、Bashターミナルをお勧めします。もちろんWindowsのコマンドプロンプトやPowerShellターミナルでも操作可能です。

Node.jsをインストール

Node.jsのWindows版をインストールします。

http://nodejs.org/

GitHub Pagesの設定

作成したリポジトリにてGitHub pagesの設定を行います。

GitHub pagesの設定方法はこちらを参照。
https://qiita.com/kanazwk/items/947db76cfa2d879fbfe0

hexoのインストール

hexoをインストールします。

リポジトリフォルダに移動し、hexo initコマンドを実行します。
その後npm installを実行します。おそらくhexoに必要な依存関係のあるパッケージを、リポジトリフォルダ配下にインストールしているようです。

ローカルでhexoサービスを起動します。ブラウザで、http://localhost:4000/を表示すると、デフォルトテーマlandscapeのサイトが表示され、動作確認ができます。今後も多用することになるコマンドの一つです。

ブログトップページの表示が確認できたら、プロンプトでCRTL + Cを実行して終了します。

GitHub Pagesを表示する

GitHub PagesのURLをブラウザで表示し、hexo server実行時と同様の表示がされればひとまず成功です。

http://<GitHubユーザ名>.github.io/

各種テーマの導入も比較的簡単で、WordPressの導入と比較するとかなり容易に導入できるのではないでしょうか。ここまで無料であり、最低限ドメインの取得費用だけで済むかと思います。

hexo ブログ構築 管理画面導入

hexo ブログの立ち上げに挑戦中。
Disqusのコメント機能にしばし奮闘したが、表示されず。icarusは対応しているようだけど、pluginがいるのだろうか。
後々調べてみよう。

今日はブラウザで編集する、管理画面を導入してみる。

hexo-adminの導入

localhost:4000/admin
で管理画面 Hexo Adminが表示される。

ただ、このままデプロイすると誰でも入れてしまうので、Hexo AdminのSettingメニューでログインアカウントを設定する。入力フォームに従って、入れると、_config.ymlに転記するコードが表示されるので、コピペしてhexo generateしておく。

これだけではGitHub pagesからのアクセスはできないようなので、ローカルでの新規作成に限られる。deployもコマンドを設定すると出来るらしい。

ちなみに手動記事はどうやって投稿するかというと、まずは以下のコマンドを実行し、_draftフォルダに生成された.mdファイルを_postフォルダに移動させるという方法のようだ。

意外と困ったのが記事の削除。
最初に作ったhello worldどう消すんだ？ということでコマンドを調べたが、投稿削除はコマンドが見当たらない。
ファイルをそのまま消してしまえばいいのかなと思い、_postフォルダ配下に一度 old フォルダを切って記事を入れてみたが、どうやら記事移動はtrackされてそのまま残るようなので、思い切ってファイルを削除したところ、hexo generateコマンドで反映された。なんともシンプルなつくりである。
いきなり削除が怖いので、本来投稿した記事を取り下げたい場合は、_draftフォルダに移動すればよい。

hexoを使ったブログ公開

about hexo

GitHub pagesへhexo 静的サイトジェネレータを使ったブログを構築した。

多分読み方はヘキソ。へくそはちょっとやめてほしい。
Node.jsベースのOSSだ。
至って導入は簡単で、各種ブログのテーマもgithub上に公開されており今回はicarusというテーマを利用した。
icarus - github

導入自体はこのサイトあたりを参考に実施。
https://qiita.com/wawawa/items/1a2f174fb29c35302543

Wordpressで動的サイトを作るのも、レンタルサーバ代やら工数やらかかりそうであるが、hexoによるブログ作成は無料であるし、Node.jsの深い知識は今のところ必要ないと思っている。（全く知識なしでここまで立ち上げることができた。）

記事投稿もマークダウンで書けるので楽そうである。
サムネイル画像の投稿がいまいちわかっていないので今後の課題。

アイコンやバナーの作成などはスマホアプリでちょちょいと作った。

明日は、Disqusというコメント機能プラグインを導入してみる予定。

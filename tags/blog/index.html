hexo ブログ構築 管理画面導入

hexo ブログの立ち上げに挑戦中。
Disqusのコメント機能にしばし奮闘したが、表示されず。icarusは対応しているようだけど、pluginがいるのだろうか。
後々調べてみよう。

今日はブラウザで編集する、管理画面を導入してみる。

hexo-adminの導入

localhost:4000/admin
で管理画面 Hexo Adminが表示される。

ただ、このままデプロイすると誰でも入れてしまうので、Hexo AdminのSettingメニューでログインアカウントを設定する。入力フォームに従って、入れると、_config.ymlに転記するコードが表示されるので、コピペしてhexo generateしておく。

これだけではGitHub pagesからのアクセスはできないようなので、ローカルでの新規作成に限られる。deployもコマンドを設定すると出来るらしい。

ちなみに手動記事はどうやって投稿するかというと、まずは以下のコマンドを実行し、_draftフォルダに生成された.mdファイルを_postフォルダに移動させるという方法のようだ。

意外と困ったのが記事の削除。
最初に作ったhello worldどう消すんだ？ということでコマンドを調べたが、投稿削除はコマンドが見当たらない。
ファイルをそのまま消してしまえばいいのかなと思い、_postフォルダ配下に一度 old フォルダを切って記事を入れてみたが、どうやら記事移動はtrackされてそのまま残るようなので、思い切ってファイルを削除したところ、hexo generateコマンドで反映された。なんともシンプルなつくりである。
いきなり削除が怖いので、本来投稿した記事を取り下げたい場合は、_draftフォルダに移動すればよい。

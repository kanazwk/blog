【心理学】敵意帰属バイアスが人間関係トラブルの原因

認知バイアス

認知バイアスとは、人の思考において必ず陥る可能性がある、不合理な選択のこと。

どういうことかというと、

  • 先入観
  • 直感
  • 恐怖
  • 願望

などによって、合理的な判断を歪めてしまうこと。

例えば、こうなったらなぁと思って計画を立てると思うが、未来予知能力を持った人間はいないので、大抵の場合、大なり小なり予想と違う結果が出る。

すると、明らかにイレギュラーな結果に対して、事前に予想した結果を継続的に期待し続けてしまう場合がないだろうか。「あの結果になるはずなのにおかしい」と、当初思い描いた結果に執着し、事実を事実として受容せず、不合理に結果を否定してしまうのだ。その結果、おもしろくない、嫌いだ、などの負の感情を抱いてしまう。

このように、思考を歪めてしまうことを認知バイアスと言う。

正常な思考を歪めてしまい、客観的な視点が抜けた主観的な思考回路によって判断の正常性が低くなる。先の例は、事前に予測した結果と異なる、つまり確証を持っていた結果と異なる結果を否定してしまう。これは認知バイアスの一種である「確証バイアス」にあたる。

認知バイアスの種類

認知バイアスは性質によって分類され、名前がついているものがある。

ハロー効果
外見が優れていると能力も高いと認識してしまうこと。

自己奉仕バイアス
自分の関わった行動が、他人の結果よりうまくいく確率が高いと認識してしまうこと。

バンドワゴン効果
食べてもいないのに行列の出来る店が美味しいと認識してしまう。

など。

誰しも持つ感覚であるので、認知バイアスを研究する事で大いにマーケティングに応用する事ができる。

例えば、バンドワゴン効果。席数を少なくし、行列を作る席数の少ないラーメン店などは、利用客の認知バイアスを2つの点で利用している。食べてもいないのに、あれだけ行列が出来ていると言うことは美味しそうだ、と認知してしまうこと。もう一つは、食べる時に、行列に並んだことによるストレスや空腹感から解放されることによって、並んでいない状態よりも幸福感が上乗せされ、味の満足度が高まるという効果。

人によっては「行列に並んだ時間の対価が味にあるはずだ」と思い込み、味にバイアスがかかる効果もあるかもしれない。

このように認知バイアスによるマーケティング戦略が、店舗の席数に反映されている。回転数の多いラーメン店ならではの手法であり、そう言った客層を無視しても利益をあげられると踏んでいる事が認知バイアス自体を理解する事によってわかる。

敵意帰属バイアスが日常に存在した

認知バイアスの一種である敵意帰属バイアスと言うものがある。

西野亮廣voicy　「コミュニケーションの心得。『敵意帰属バイアス』を理解しろ
」より。

https://voicy.jp/channel/941/64714?playlistId=64714&referrer=https:%2F%2Fnote.com%2Fk_kana%2Fn%2Fn300bba664835&speakerId=3757

文章と件のポスターは西野さんの過去ブログでも掲載されている。

西野亮廣『話が進まない時に潜む「敵意帰属バイアス」』
https://ameblo.jp/nishino-akihiro/entry-12367497785.html

この話を聞いて、これだったのか！と気づいた。

身の回りで思い当たる節が多すぎた。自分の身に降りかかった過去の否定的な対応の数々や、話に聞く話もこのバイアスによって歪められた結果だと思うと、大変な腹落ち感があった。

同じチームで仕事をする以上、ぶつかる事も多いが、あきらかに感情的に否定される事が多い人間がいる。話すたびに理論的な問答ではなくなって行き、終いには何故か怒ったり、意見を言うと拗ねられてしまい、正常なコミュニケーションが取れないのだ。

敵意帰属バイアスであると仮定するとかなりハマる事が多く、こちらが理論的に正しい事を言ったところで、馬鹿にされている、否定されていると曲解されてしまう。相手にしては私の意見が自分と異なれば異なるほど面白くなく感じ、怒るか拗ねるかされた挙句、話が進まず、仕事が遅れるという訳だ。

経験上、そう言う人間に限って仕事能力が低い事が多い。会話する前段階で敵意帰属バイアスが発動する事もあるようで、否定される事を恐れて相談や報告もして来なくなる。

結果、仕事がさらに遅れるのだが、それを注意すると更に敵意を剥き出しにしてくるため、余計に拉致が開かない。

この無限ループにハマった事が何度かあり、結果彼らの尻拭い業務が山積みになる。採用やチームビルディングの段階でなかなか気づけないのがこの属性所持者であったが、今後このバイアス属性の高さを初期段階で見抜くよう慎重に質疑応答するよう、採用等に取り入れて行きたい。

そのような相手の考え方を変えようとしても、その努力は徒労に終わる事が多かろうと思う。他人と過去は変えられないので、自分と未来を変えて行こう。

あなたの周りにも、こちらの意見を曲解しがちな人物がいたら、敵意帰属バイアスに支配されている可能性がある。対処法としては、相手にしない事だが、相手にせざるを得ない状況にある事も多いだろうと思う。一旦距離を置く事が望ましいとされるので、思い切って環境を変えるなど、勇気ある行動を取りたい。

他人の認知の歪みに振り回されて、自分の心の健康を損なわぬよう気をつけよう。自分にも言い聞かせるつもりで、本日の結びとする。

歴史上の偉人の業績と年齢

偉業を達するのに年齢は関係ない。そう言わんばかりに歴史上の人物は短命な事が多い。偉業を達成した年齢も今の世で考えると驚異の若さだ。

大政奉還

坂本龍馬は33歳の若さで暗殺された。

江戸幕府15代将軍、徳川慶喜が政権を朝廷に委譲した大政奉還の1ヶ月後の事であった。

明治維新は日本国民がほぼ知るところの革命であり、革命活動の中心でにあった坂本龍馬は男児の憧れの人物である事は言わずもがな。

剣術の腕は北辰一刀流の免許皆伝、さらに亀山社中と言う日本初の商社を起業するビジネスの腕も持ち合わせる。結婚し、新婚旅行も楽しんでいたそうだ。

龍馬の成した偉業は日本史を大きく転換させる事になるが、その短い人生で圧倒的な行動力を示した。

当時の生きる速度と比較すると、段違いに現代が遅く感じるのは何故だろう。

桶狭間の戦い

織田信長が今川義元を桶狭間の戦いで破ったのは、若干齢26の歳の事だった。兵力の差は、今川軍2万5000に対し、織田軍3000。

兵の士気を上げるため、早朝4時に5名の兵のみ率いて熱田神宮へ出陣した信長。殿を守るべしと追って馳せ参じた兵は俄然士気が上がったそうだ。戦略的で見事なリーダーシップである。

当日豪雨が襲った桶狭間にて、義元は近衛兵のみ従えて雨を回避するよう本軍を離れた。そこに勝機を見逃さぬ行軍で義元の首級をあげた戦術は、兵力差をひっくり返す見事な決断力だ。

現代も若者達は商機を逃さぬよう行動しているだろうか。

松下村塾

幕府軍を討ち破った高杉晋作や、明治政府の首脳である伊藤博文、山縣有朋など後の歴史上の偉人を輩出した松下村塾。

吉田松陰は享年29歳。松下村塾を先代から引き継いだのが、27歳。

松蔭をはじめ多くの塾生が若くして幕末の世に散り、歴史に名を残した。高杉晋作も享年27歳の若さだ。

数百年後、同じ思想のもとに時代を変えたコミュニティとして語られる団体が、現代にはいくつ浮かぶだろうか。

芸術家にも

よく知る歴史上の人物は、現代から比べると驚くべき若い年齢で偉業を達成している事がわかる。

幕末→戦国→幕末と事例を紹介してしまったが、芸術家も短命な人物が多い。

小説家、樋口一葉は享年24歳。

作曲家、滝廉太郎は23歳。

海外の芸術家で言えば、画家のゴッホは享年37歳。

音楽家では、シューヴェルトが享年31歳。モーツァルトは享年35歳だ。日本の尾崎豊は享年26歳の若さだ。

いつに間にか、教科書や伝記で読んだ歴史上の偉人達の年齢を追い越してしまった。現代、平均寿命が数十年伸びたとは言え、名前の刻むべきところはどこか、男子たるもの考えてしまう。

hexoを使ったブログ公開

about hexo

GitHub pagesへhexo 静的サイトジェネレータを使ったブログを構築した。

多分読み方はヘキソ。へくそはちょっとやめてほしい。
Node.jsベースのOSSだ。
至って導入は簡単で、各種ブログのテーマもgithub上に公開されており今回はicarusというテーマを利用した。
icarus - github

導入自体はこのサイトあたりを参考に実施。
https://qiita.com/wawawa/items/1a2f174fb29c35302543

Wordpressで動的サイトを作るのも、レンタルサーバ代やら工数やらかかりそうであるが、hexoによるブログ作成は無料であるし、Node.jsの深い知識は今のところ必要ないと思っている。（全く知識なしでここまで立ち上げることができた。）

記事投稿もマークダウンで書けるので楽そうである。
サムネイル画像の投稿がいまいちわかっていないので今後の課題。

アイコンやバナーの作成などはスマホアプリでちょちょいと作った。
